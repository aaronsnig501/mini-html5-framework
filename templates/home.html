What is mini-html5-framework?

And why do we need yet another JavaScript framework?

mini-html5-framework is an incredibly lightweight framework that comes with little more than a simple means of routing with basic templating and controller support. While most JavaScript frameworks attempt to enhance the user experience for large scale webapps, not all projects require the bloat of a framework like Angular, it's nice to provide the same user experience of a Single Page Application of smaller applications like blogs.

How does it work?

mini-html5-framework is built on top of the HTML5 History API.
